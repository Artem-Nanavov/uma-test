# Для быстрого старта проекта

### Быстрый старт

Install dependencies

```
npm i
```

Run development build

```
npm run dev
```

Run production build

```
npm start
```

### Команды

- npm run dev
- npm run build
- npm start
- npm run lint
- npm run test
